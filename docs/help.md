## 1. 快速开始

1. 启动 My Excel。
2. 菜单栏 **文件 → 导入**，选择 `.xlsx` 文件。
3. 文件加载后，表格内容即显示在中央区域。
4. 点击单元格查看/编辑内容，按 **Ctrl+S** 保存。

---

## 2. 表格查看与虚拟滚动

### 核心能力

- **虚拟渲染**：仅绘制当前视口内可见的单元格，支持万行级大表格流畅滚动。
- **双向滚动**：鼠标滚轮/拖拽滚动条，支持水平和垂直双向滚动。
- **单元格渲染**：完整支持 Excel 单元格属性——值/公式、背景色、字体大小/颜色/加粗、9 种对齐方式、数字格式（日期自动转换）、单元格边框。
- **合并单元格**：自动识别与正确渲染，选中时"吸附"到合并区域边界。
- **批注指示器**：有批注的单元格右上角显示橙红色三角标记，悬停显示作者和批注正文。

### 操作方式

| 操作 | 说明 |
|------|------|
| 鼠标滚轮 | 垂直滚动 |
| Shift + 滚轮 | 水平滚动 |
| 拖拽滚动条 | 双向滚动 |
| 滚轮点击 | 自动滚动 |

## 3. 单元格导航与选择

### 键盘导航

| 快捷键 | 行为 |
|--------|------|
| `↑` `↓` `←` `→` | 向对应方向移动一个单元格（越界自动滚动） |
| `Tab` | 向右移动（行末跳到下一行首列） |
| `Shift+Tab` | 向左移动（行首跳到上一行末列） |
| `Enter` | 进入编辑模式（编辑中按 Enter 保存退出） |
| `Ctrl+A` | 编辑模式下全选输入框文本 |

> 合并单元格导航：方向键跳转到合并区域左上角/边界，自动整体跳格。

### 鼠标选择

| 操作 | 行为 |
|------|------|
| **左键单击** | 选中单个单元格（绿色边框），同时更新 Shift+点击锚点 |
| **Shift + 左键单击** | 从锚点到目标格的矩形范围选择（蓝色半透明填充） |
| **左键拖拽** | 按住左键拖拽选择矩形区域 |
| **左键双击** | 进入该单元格编辑模式 |
| **右键单击** | 选中单元格并打开右键菜单 |

### Shift+点击范围选择（与 Excel 一致）

1. 单击单元格 A2 设为活动格（自动设为锚点）。
2. 按住 `Shift`，单击 C5 → 选中 A2:C5 范围。
3. 继续按住 `Shift`，单击 D3 → 范围更新为 A2:D5。
4. Shift+点击锚点本身（A2）→ 退回单格选中。

> 活动单元格（绿色边框）始终保持不变；方向键/Tab 导航也会更新锚点。

## 4. 单元格编辑

### 进入编辑模式

| 方式 | 操作 |
|------|------|
| **双击** | 双击目标单元格 |
| **Enter** | 选中单元格后按 Enter |
| **键入即编辑** | 选中单元格后直接键入字符（自动替换原内容） |
| **公式栏** | 点击公式栏输入框，输入后按 Enter 写入选中格 |

### 编辑中操作

| 快捷键 | 行为 |
|--------|------|
| `Enter` | 保存编辑并退出 |
| `Escape` | 取消编辑，恢复原始值 |
| `Ctrl+A` | 全选编辑框文本 |
| `Ctrl+Z` | 撤销编辑框内文本（不影响表格撤销栈） |
| 点击单元格外部 | 保存编辑并退出 |

### 编辑特性

- **原位编辑**：直接在单元格内编辑，无额外输入框。
- **公式识别**：以 `=` 开头的内容自动识别为公式。
- **实时重算**：编辑公式时依赖单元格即时更新。
- **日期转换**：日期格式单元格编辑时自动在日期字符串与序列号之间转换。
- **数据有效性校验**：非公式值保存时自动校验，失败弹出错误提示。

## 5. 复制与粘贴

### Ctrl+C 复制

1. 选中单个单元格或矩形区域（点击/Shift+点击/拖拽）。
2. 按 `Ctrl+C`。
3. 内容以 **TSV 格式**（Tab 分隔列，换行分隔行）写入**系统剪贴板**，因此可粘贴到其他程序。
4. 公式保留 `=` 前缀，日期转为格式化日期字符串。

### Ctrl+V 粘贴

1. 选中目标起始单元格。
2. 按 `Ctrl+V`。
3. 直接从**系统剪贴板**读取内容（无论内容来自本程序的 Ctrl+C，还是从其他程序复制），因此支持跨应用粘贴。
4. 内容按 TSV 解析，从起始格开始写入。

> **编辑文本时的复制/粘贴**：当焦点在公式栏、名称框或正在原位编辑单元格时，`Ctrl+C`/`Ctrl+V` 作用于正在编辑的文本本身（选中文字复制 / 在光标处粘贴），而非整格——与 Excel 行为一致。

| 粘贴场景 | 行为 |
|----------|------|
| 单格 → 单格 | 直接覆盖目标格 |
| 多格 → 单格起始 | 按原行列结构写入（起始格为左上角） |
| 含 `=` 的值 | 识别为公式写入 |
| 粘贴后选区 | 多行/多列粘贴自动更新选中范围 |

> 粘贴操作**可撤销**（`Ctrl+Z`）。

## 6. 拖拽选择与填充柄

### 拖拽框选

按住左键在表格上拖拽，选中矩形范围（蓝色半透明填充 + 绿色边框）。

### 填充柄（Fill Handle）

选区右下角的深灰色小方块：

1. 选中一个或多个单元格。
2. 将鼠标移到右下角的深灰色方块（光标变为十字 `+`）。
3. 按住左键向目标方向拖拽。
4. 释放鼠标，数据自动填充到目标区域。

| 源数据类型 | 填充行为 |
|------------|----------|
| **数字序列**（1, 2, 3） | 按步长递增（+1） |
| **等比序列**（2, 4, 8） | 按比值递增（×2） |
| **日期**（2026/1/1, 1/2） | 按天递增 |
| **文本** | 循环复制 |
| **公式** | 相对引用自动平移（`$` 绝对引用不变） |

> 填充操作**可撤销**（`Ctrl+Z`）。

## 7. 右键上下文菜单

在表格单元格上右键单击，弹出上下文菜单：

| 菜单项 | 说明 |
|--------|------|
| **在上方插入行** | 在当前行上方插入指定行数（可调） |
| **在下方插入行** | 在当前行下方插入指定行数（可调） |
| **在左侧插入列** | 在当前列左侧插入（需确认弹窗选择复制选项） |
| **在右侧插入列** | 在当前列右侧插入（需确认弹窗选择复制选项） |
| **清空单元格 / 清空选中范围** | 清除选中格的值和公式（需确认） |
| **向下选中** | 从当前格向下选中到边界/指定行数 |
| **向上选中** | 从当前格向上选中 |
| **向左选中** | 从当前格向左选中 |
| **向右选中** | 从当前格向右选中 |

> 插入行/列操作**可撤销**（全量快照级别）。

## 8. 搜索与筛选

### 入口

菜单栏 **搜索 → 搜索**，打开非模态搜索窗口。

### 列筛选

1. 在搜索窗口的**列**页签中，从下拉框选择目标列。
2. 输入搜索关键字，选择比较运算符（包含/不包含/等于/不等于/大于/小于/大于等于/小于等于）。
3. 点击**搜索**，不匹配的列被隐藏（视觉上"折叠"）。
4. 支持**多条件列筛选**：添加多条筛选条件，选择 And（交集）或 Or（并集）组合逻辑。

### 行筛选

1. 切换到**行筛选**页签。
2. 配置行筛选规则（列、关键字、运算符、组合逻辑）。
3. 点击**搜索**，不匹配的行被隐藏。

### 搜索配置

菜单栏 **配置 → 搜索配置**，可持久化列筛选/行筛选的输入范围配置到 YAML 文件（`~/.MyExcel/my-excel.yaml`），下次打开自动加载。

### 操作说明

- **重置搜索**：关闭搜索窗口或切换 Sheet 时自动恢复所有隐藏行列。
- **性能**：大表格（>1000 行）自动启用多线程并行扫描。

## 9. 冻结窗格

冻结窗格由 Excel 文件自身定义（`frozen_rows` / `frozen_cols`），打开即生效。

### 视觉效果

- **顶部冻结行**：固定在视口顶部（如列标题行），上下滚动时不移动。
- **左侧冻结列**：固定在视口左侧（如行号列），左右滚动时不移动。
- **冻结分隔线**：2px 深灰色实线，区分冻结区域与滚动区域。
- **左上角**：冻结行与冻结列交叉区域（始终固定）。

### 交互

- 冻结区域内的单元格可正常点击/编辑。
- 拖拽选择和 Shift+点击在冻结区域内使用**视口相对坐标**，不受滚动影响。

## 10. 公式系统

### 支持的运算符

| 类型 | 运算符 |
|------|--------|
| 算术 | `+` `-` `*` `/` `^` `%` |
| 比较 | `=` `<>` `<` `<=` `>` `>=` |
| 文本连接 | `&` |

### 支持的函数

| 函数 | 说明 | 示例 |
|------|------|------|
| `SUM` | 求和 | `=SUM(A1:A10)` |
| `AVERAGE` | 求平均值 | `=AVERAGE(B1:B5)` |
| `COUNT` | 计数（仅数值） | `=COUNT(A1:A100)` |
| `MAX` | 最大值 | `=MAX(C1:C20)` |
| `MIN` | 最小值 | `=MIN(D1:D10)` |
| `IF` | 条件判断 | `=IF(A1>60,"及格","不及格")` |
| `CONCATENATE` | 文本连接 | `=CONCATENATE(A1," ",B1)` |

### 引用格式

| 类型 | 格式 | 示例 |
|------|------|------|
| 相对引用 | `A1` | 填充/复制时自动调整行列号 |
| 绝对引用 | `$A$1` | 行列号固定不变 |
| 混合引用 | `$A1` 或 `A$1` | 列或行一方固定 |

### 公式特性

- **实时重算**：编辑保存后立即计算所有依赖公式。
- **增量求值**：值变更仅重算受影响的公式（`evaluate_dependents`），不触发全表重算。
- **循环依赖检测**：拓扑排序自动处理。
- **错误值**：`#DIV/0!`（除零）、`#VALUE!`（类型错误）等。

## 11. 条件格式

条件格式规则从 Excel 文件中读取并应用，在值变化时自动重新计算。

### 规则字段

| 字段 | 说明 | 示例 |
|------|------|------|
| 规则（运算符） | 条件运算符 | `>` `<` `=` `>=` `<=` `!=` |
| 值 | 比较阈值 | `60`、`充足` |
| 填充色 | HEX 颜色 | `FFC7CE`、`#FFC7CE` |
| 应用于 | 应用范围 | `=G3:G154`、`=B7:B~`（动态延伸） |

### 应用范围语法

- **静态范围**：`=G3:G154`（固定区域）、`=$G$3:$G$154`（`$` 自动忽略）
- **动态范围**：`=B7:B~`（向下延伸到数据边界）、`=B7:~7`（向右延伸）、`=A1:~`（全表）

> 条件格式随公式求值结果动态更新（`cf_dirty` 标志位驱动）。

## 12. 数据预警

### 入口

菜单栏 **配置 → 预警消息**，打开预警规则管理弹窗。

### 预警规则

每条规则包含：
- **运算符**：比较运算符
- **阈值**：比较值
- **预警消息**：触发时显示的提示文本
- **应用范围**：单元格范围（插入行列时自动扩展偏移量）

### 预警通知

- 菜单栏右侧的 **🔔 预警图标** 显示未读预警数量。
- 点击图标展开预警通知面板，逐条显示触发的预警消息。
- 规则持久化到 YAML 文件（`~/.MyExcel/my-excel.yaml` 的 `alert` 节点）。

## 13. 转换工具

### 入口

菜单栏 **转换 → 转换工具**，打开转换弹窗。

### 功能

根据用户定义的转换规则，将源表单元格按规则映射到目标位置，生成新的 Excel 文件。

### 规则语法

基本格式：
```
源范围 -> 目标起始位置 : 方向标记 ~ ;
```

| 符号 | 含义 |
|------|------|
| `|` | 纵向填充 |
| `-` | 横向填充 |
| `~` | 按源单元格数量等量填充 |
| `(N1:O1)` | 合并单元格对 |
| `(A:M)3:(A:M)12` | 批量列范围 |
| `(N3+2)` | 步长跳跃取值 |
| `,formula(cell=expr)` | 自定义公式（`~` 为动态行尾占位符） |

> 详细规则示例见菜单栏 **关于 → 帮助**。

## 14. 撤销 / 重做

### 撤销操作

**`Ctrl+Z`**（不含 Shift）撤销上一步操作。

### 撤销粒度

| 操作类型 | 粒度 | 说明 |
|----------|------|------|
| 插入行/列 | **全量快照** | 恢复整张表数据 |
| 单元格编辑/清空 | **单格变更** | 恢复单个单元格旧值 |
| 范围清空 | **范围清空** | 恢复范围内所有单元格旧值 |
| 填充柄填充 | **范围清空** | 恢复被覆盖的目标格旧数据 |
| Ctrl+V 粘贴 | **范围清空** | 恢复被覆盖的目标格旧数据 |

### 限制

- 撤销栈深度上限 **20 步**（FIFO，超出丢弃最旧记录）。
- 仅支持撤销（`Ctrl+Z`），暂不支持重做（`Ctrl+Y`）。
- 编辑模式下 `Ctrl+Z` 交给输入框做文本内撤销，不影响表格撤销栈。

## 15. 文件导入与保存

### 导入

1. 菜单栏 **文件 → 导入**，弹出文件选择对话框。
2. 选择 `.xlsx` 文件。
3. 异步加载（后台线程），加载期间显示 spinner 动画。
4. 加载成功：替换当前数据，重置选中/撤销/隐藏集合，应用条件格式与预警规则。
5. 加载失败：显示错误信息。

### 保存

| 方式 | 操作 |
|------|------|
| **Ctrl+S** | 保存快捷键 |
| **💾 保存按钮** | 名称栏右侧（脏标记时蓝色高亮） |

### 保存行为

- 直接**覆盖写入原文件**（不另生成带日期后缀的新文件）。
- 异步保存（后台线程），保存期间显示 spinner。
- 保存成功：状态栏右侧以绿色显示原文件路径，脏标记清除。
- 保存失败：弹出居中红色提示框（"知道了"关闭），常见原因为原文件被其它程序占用打开。

## 17. 名称框与公式栏

名称框行位于菜单栏下方，从左到右排列：

```
┌──────────┬───┬──────────────────────────────────┬──────────┐
│ 名称框    │ ▼ │ fx │  公式输入框                  │  💾 保存  │
│ (跳转)    │   │   │  (Enter 写入选中格)           │          │
└──────────┴───┴──────────────────────────────────┴──────────┘
```

### 名称框

- 输入单元格引用（如 `A1`、`AA100`）按 Enter → 跳转到该单元格。
- 下拉箭头显示"定义名称"/"管理名称"入口。

### 公式栏

- 自动显示当前选中单元格的值或公式。
- 可直接编辑，按 Enter 写入选中格（触发校验与重算）。

## 18. 数据有效性

### 输入提示

选中带有数据有效性输入提示（Input Message）的单元格时，左下方自动弹出淡黄色提示框，显示提示标题和内容。

### 校验错误

编辑保存时，若值不符合数据有效性规则：

1. 弹出固定位置的黄色错误弹窗（标题 + 错误消息）。
2. **重试**：关闭弹窗，可修改值后重新提交。
3. **取消**：恢复编辑前的原始值。
4. 校验弹窗显示期间，**锁定所有表格交互**（键盘、点击、拖拽均不可操作）。

## 19. 配置持久化

### 配置文件位置

`~/.MyExcel/my-excel.yaml`

### 配置项

| 节点 | 内容 |
|------|------|
| `insert.column` / `insert.row` | 插入列/行的合并配置与复制选项 |
| `search.column` / `search.row` | 搜索列筛选/行筛选的输入范围 |
| `alert` | 预警消息规则列表 |

### 入口

- **配置 → 搜索配置**：编辑搜索配置并保存到 YAML。
- **配置 → 预警消息**：编辑预警规则并保存到 YAML。

## 20. 授权与激活

### 授权状态

| 状态 | 说明 |
|------|------|
| 试用中 | 功能正常使用，菜单栏「关于」显示剩余天数 |
| 已授权（永久/有限期） | 全部功能可用 |
| 试用期已结束 | 全屏拦截遮罩，无法使用 |
| 授权已到期 | 全屏拦截遮罩 |
| 授权异常 | 全屏拦截遮罩 |

### 激活

菜单栏 **关于 → 激活**，在试用期内可提前激活。
